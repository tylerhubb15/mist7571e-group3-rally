RALLY — PRESENTATION DOCS
MIST 7571E | Group 3
============================================================

Each team member: add your elevator pitch as a .txt file
to this folder before the presentation.

FILE NAMING
-----------
Use this format:  YourName_RoleX_pitch.txt

Examples:
  Tyler_Role2_pitch.txt
  Alex_Role1_pitch.txt
  Jordan_Role3_pitch.txt

WHAT TO INCLUDE IN YOUR .txt FILE
-----------------------------------
Keep it to 1 minute 30 seconds or less when spoken out loud.
That is roughly 4-5 short sentences. Cover:

  1. Your role name and what it meant for the project
  2. The main thing(s) you built or configured
  3. Why your piece matters to the app or the user
  4. A one-sentence handoff to the next person

TIPS
----
- Write it the way you would say it, not how you would write an essay
- Read it out loud and time yourself before the presentation
- If it takes more than 90 seconds to say, cut something
